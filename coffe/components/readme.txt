Директория с компонентами.
Для совместимости с ОС семейства UNIX желательно указывать имена компонентов (директории) в нижнем регистре.
Символ точка используется для привязки компонентов к модулям (также это позволяет визульно группировать компоненты в директории). 
Например, main.fullmenu означает, что данный компонент не будет запущен без модуля main.
Класс компонента в этом случае должен называться Coffe_Main_Fullmenu.
